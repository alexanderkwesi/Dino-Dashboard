# Dino Dashboard

An interactive dashboard-style project built around a small Dino game concept, with focus on state, feedback and presentation.

## Why I Built It

I built this project to practise turning a small interactive idea into something more presentable and measurable through dashboard-style framing.

## Features

- Interactive game/dashboard concept
- Screenshot evidence in the repository
- Focus on UI feedback and presentation
- Lightweight project suitable for portfolio review
